# Google Calendar Event Automation

Automated calendar management system using Python and Playwright to create recurring Google Calendar events from CSV data.

## Features
- Automated event creation from CSV
- Dynamic form filling
- Custom recurrence patterns
- Calendar categorization

## Technologies
- Python 3.x
- Playwright
- CSV processing

## Installation
```bash
pip install playwright
playwright install chromium
```

## Usage
1. Prepare your CSV file with columns: Class Name, Code, Start Time, End Time, Room, Day1, Day2
2. Run: `python main.py`
3. Login to Google Calendar (first time only)

## Impact
Reduced manual calendar entry from hours to minutes.
